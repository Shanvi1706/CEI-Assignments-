# CEI Assignment 1 - Data Cleaning using Pandas

## Objective
Perform data exploration and data cleaning using Python Pandas.

## Tasks Performed
- Loaded CSV dataset into Pandas DataFrame
- Explored dataset using head(), tail(), shape, columns, and info()
- Checked and handled missing values
- Removed duplicate rows
- Performed filtering and column selection
- Created derived column `total_amount`
- Saved cleaned dataset as CSV
